インストール¶

Choreonoid Editorプラグインをインストールするには、まずChoreonoid本体をインストールする必要があります。

Choreonoidエディタをppaからインストールする（推奨）¶

Choreonoidはppaからインストールすることをおすすめします。

$ sudo add-apt-repository ppa:hrg/daily
$ sudo apt-get update
$ sudo apt-get install choreonoid-editor

Choreonoidエディタをソースからインストールする¶

Choreonoidをppaからインストールする¶

Choreonoidはppaからインストールすることをおすすめします。

$ sudo add-apt-repository ppa:hrg/daily
$ sudo apt-get update
$ sudo apt-get install choreonoid libcnoid-dev

Choreonoidをソースからインストールする¶

警告

ソースコードからのインストールは上級者向けの方法です。ppaからのインストールを強く推奨します。

Choreonoidをソースからインストールしたい場合、以下の手順に従ってください:

日本語での説明:
http://choreonoid.org/ja/manuals/1.5/install/build-ubuntu.html
英語での説明:
https://github.com/s-nakaoka/choreonoid/blob/master/INSTALL

Choreonoid Editorプラグインのインストール¶

githubから最新のソースコードをクローンします。

$ git clone https://github.com/fkanehiro/choreonoid-editor.git

次にcmakeを実行します。

$ cd choreonoid-editor
$ cmake .

コンパイル、インストール、実行するために以下のコマンドを実行してください。

$ make
$ sudo make install
$ choreonoid
